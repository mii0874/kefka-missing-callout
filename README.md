# ケフカ ミッシング コールアウト

FF14「絶妖星乱舞」のP2ミッシングギミック用の音声コールアウトツールです。

ブラウザの音声合成を使って、タイムラインに沿ってギミックのタイムラインを読み上げます。

---

## 使い方

### 起動方法

- **ブラウザで開く場合：** [GitHub Pages](https://mii0874.github.io/kefka-missing-callout/callout.html) にアクセス
- **ローカルで使う場合：** `callout.html` をダウンロードして、ファイルをダブルクリック

### モード選択

| モード | 説明 |
|--------|------|
| 🧪 テストモード | 開始ボタンを押すと即座に 0s からスタート。動作確認に。 |
| 🔴 本番モード | カウントダウン後、ミッシング開始タイミングまで待機してからスタート。 |

### 本番モードのカウントダウン秒数

本番モード時に表示されるプルダウンで、**開始ボタンを押すタイミング**を調整できます（1〜21秒）。

- ミッシング詠唱開始の何秒前に押すかに合わせて設定してください
- デフォルトは **5秒**

### 操作ボタン

| ボタン | 説明 |
|--------|------|
| ▶ 開始 | タイマースタート |
| `<<` | 0.5秒戻す（押しっぱなしで連続） |
| `>>` | 0.5秒進める（押しっぱなしで連続） |
| ⏸ 一時停止 | タイマーを一時停止 |
| ↺ リセット | 最初に戻す |

> 押し遅れたときは `>>` を押して調整してください。

## コールアウト内容のカスタマイズ

`callout.html` をテキストエディタで開き、`const EVENTS` の部分を編集します。

```javascript
const EVENTS = [
  { at:   0, label: 'ミッシング開始' },
  { at:   4, label: '1回目' },
  { at:  15, label: '2回目' },
  // ...
];
```

- **`at`** — ミッシング開始（0s）を基準にした秒数
- **`label`** — 読み上げるテキスト

### 待機時間の調整

本番モードの「ミッシング開始まで何秒待つか」は `PRE_OFFSET` で設定しています。

```javascript
const PRE_OFFSET = 235; // 3分55秒
```

## 動作環境

- Chrome / Edge 推奨（日本語音声合成の精度が高い）
- Safari / Firefox でも動作しますが、声質が異なる場合があります
- インターネット接続不要（ローカルでも動作）
